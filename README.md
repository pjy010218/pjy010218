## 🚀 Key Projects & Achievements

### 🛡️ Cyber Deception & Anti-Tampering
* **IaC 기반 사이버 기만 체계**: 인프라 코드(IaC)를 반영하여 시스템과 동기화되는 자가 동기화 기만 프레임워크 연구 (IEEE PRDC).
* **MP FUSE**: 리눅스 FUSE를 활용하여 파일 시스템 레벨에서 코드 변이(Metamorphic/Polymorphic)를 수행하는 기술 개발.
* **VFS 탐지 기반 안티탬퍼링**: 가상 파일 시스템(VFS) 분석을 통해 커널 수준의 위변조를 결정론적으로 식별.

### 🤖 LLM & Intelligence Security
* **LLM DLP Framework**: LLM 에이전트 환경의 데이터 유출 경로 분석 및 보안 게이트웨이 기술 동향 연구.
* **AORM Engine**: 공격 행위와 객체 간의 관계를 수치화하여 침해 사고의 내부 파괴력을 평가하는 리스크 매트릭스 설계.
* **Shisa-Kanko (Code Observer)**: AI 에이전트의 코드 수정안을 실시간 시각화하는 VS Code Extension 배포.

### 🛠️ Systems Engineering
* **eBPF-based Drone Security**: eBPF를 이용한 커널 패킷 인터셉션 및 드론 스웜용 킬스위치 시스템 설계(특허 진행 중).
* **AI Barista / Computer Vision Scanner**: 하드웨어 제어 및 실시간 재질 분류를 결합한 임베디드 AI 프로젝트.
